# 完形逻辑关系词分类

## 1. 转折关系 (Transition)

- **句间转折**
  however (然而) / nevertheless (不过) / nonetheless (尽管如此)
  conversely (相反地) / instead (反而)
  fortunately (幸运的是) / unfortunately (不幸的是)
  but (但是) / yet (可是)
- **句内转折**
  but (但是) / and yet (然而)

## 2. 对比对立关系 (Contrast)

- **句内对比**
  while (而) / whereas (然而) / but (但是)
  against (与...相反) / unlike (不像) / instead of (而不是)
  rather than (而非) / different from (不同于) / contrary to (与...相悖)
- **句间对比**
  by contrast (相比之下) / conversely (相反地) / on the contrary (恰恰相反)

## 3. 让步关系 (Concession)

- **句内让步**
  despite (尽管) / in spite of (虽然) / regardless of (不论)
  although (虽然) / though (尽管) / even though (即使) / even if (即使)
- **句间让步**
  In principle (原则上) / To be fair (公平地说) + A. But B.

## 4. 并列关系 (Coordination)

- **句内并列**
  and (和) / or (或) / not only...but also (不仅...而且)
  both...and (既...又) / as well as (以及) / either...or (要么...要么)
  neither...nor (既不...也不)
- **句间并列**
  moreover (此外) / furthermore (而且) / further (另外)
  also (也) / besides (此外) / additionally (另外) / in addition (除此之外)
  equally (同样地) / likewise (类似地) / similarly (类似地)
  simultaneously (同时) / meanwhile (与此同时)

## 5. 递进与强调 (Progression & Emphasis)

- **递进/强调**
  indeed (事实上) / in fact (实际上) / in effect (实质上)

## 6. 因果关系 (Cause-Effect)

- **原因**
  because (因为) / as (由于) / since (既然) / now that (既然)
  seeing that (鉴于) / considering that (考虑到) / given that (考虑到)
  due to (由于) / because of (因为) / owing to (由于)
  for the sake of (为了) / on account of (基于) / for (因为)
- **结果**
  therefore (因此) / eventually (最终) / consequently (所以)
  thus (因而) / hence (于是) / accordingly (因此)
  in consequence (结果) / as a result (因此) / so (所以)

## 7. 举例关系 (Exemplification)

- **举例**
  for example (例如) / for instance (比如) / such as (诸如)

## 8. 条件关系 (Condition)

- **条件**
  if (如果) / supposing (假设) / providing (只要)
  provided (只要) / assuming (假设) / unless (除非)
  until (直到) / as long as (只要)
