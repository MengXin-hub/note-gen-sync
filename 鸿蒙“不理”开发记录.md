# From Kit（卡片服务开发）

formProvider 模块提供了获取卡片信息、更新卡片、设置卡片更新时间等能力。

## 实现卡片交互

### 三类交互事件

动态卡片可使用 `postCardAction` 接口，静态卡片通过 `FormLink` 接口实现页面交互功能。两个接口均支持 `router`、`message` 和 `call` 三种类型的事件，事件使用场景：

- **router 事件**：可通过 `router` 事件跳转至指定 UIAbility（页面，可以理解成一个独立的用户界面功能模块），实现点击卡片跳转到该应用内对应页面的功能。对于非系统应用（自己开发的第三方应用），仅支持跳转至自身应用内的 UIAbility。
- **call 事件**：可通过 `call` 事件将指定 UIAbility 切换到后台运行，再通过 UIAbility 申请对应后台长时任务以实现音乐播放、下载、导航等功能。
- **message 事件**：可通过 `message` 事件拉起 `FormExtensionAbility`（卡片后台逻辑处理模块），`FormExtensionAbility` 利用 `onFormEvent` 接口接收到这条消息，并执行对应操作，实现点击卡片控件后向应用传递消息的功能，如接收到“暂停音乐”，调用暂停播放的代码。

### 要点总结

1. 使用**router事件**跳转到应用界面：在动态卡片中使用 `postCardAction` 接口的 `router` 能力，可快速拉起动态卡片提供方应用的指定UIAbility，因此UIAbility较多的应用通常会通过卡片提供不同跳转按钮，实现一键直达的效果；
2. 使用**router事件**，点击卡片可拉起对应应用的UIAbility至**前台**，并刷新卡片。使用**call事件**，点击卡片可拉起对应应用的UIAbility至**后台**，并刷新卡片。在卡片页面中可以通过 `postCardAction` 接口触发**router事件**或者**call事件**拉起UIAbility，然后由UIAbility刷新卡片内容；
3. 许多应用希望借助卡片的能力，实现和应用在前台时相同的功能，在卡片中使用 `postCardAction` 接口的**call能力**，能够将卡片提供方应用的指定的UIAbility拉到后台。同时，call能力提供了调用应用指定方法、传递数据的功能，使应用在后台运行时可以通过卡片上的按钮执行不同的功能；
4. 在卡片页面中可以通过 `postCardAction` 接口触发**message事件**拉起 `FormExtensionAbility`，通过 `onFormEvent` 接口回调通知，以完成点击卡片控件后传递消息给应用的功能，然后由 `FormExtensionAbility` 刷新卡片内容。

## 在应用内拉起卡片管理添加到桌面

应用通过 `formProvider` 模块提供的 `openFormManager` 接口，能够拉起卡片管理页面，触发卡片添加至桌面操作，点击“添加至桌面”，实现卡片添加至桌面的功能。

```ts
// entry/src/main/ets/pages/Index.ets

import { formProvider } from '@kit.FormKit';
import { BusinessError } from '@kit.BasicServicesKit';
import { Want } from '@kit.AbilityKit';

@Entry
@Component
struct Index {

  build() {
    Row() {
      Column() {
        //添加拉起卡片管理页面按钮，点击按钮后调用openFormManager方法，拉起卡片管理页面
        Button('拉起卡片管理页面')
          .onClick(() => {
            const want: Want = {
              bundleName: 'com.example.formbutton',
              abilityName: 'EntryFormAbility',
              parameters: {
                'ohos.extra.param.key.form_dimension': 2,
                'ohos.extra.param.key.form_name': 'widget',
                'ohos.extra.param.key.module_name': 'entry'
              },
            };
            try {
              formProvider.openFormManager(want);
            } catch (error) {
              console.error(`catch error, code: ${(error as BusinessError).code}, message: ${(error as BusinessError).message})`);
            }
          })
          .margin({ top: 10, bottom: 10 })
      }
      .width('100%')
    }
    .height('100%')
  }
}
```
